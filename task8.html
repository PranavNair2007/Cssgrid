Task Question and Answer

What problem does auto-fit solve?

It automatically adjusts number of columns based on screen width.

Why is minmax important?

It sets minimum and maximum size so it prevents items from becoming too small.

Do items shrink too much?

No, minmax stops shrinking after minimum value.

Is this layout mobile friendly?

Yes, columns can reduce automatically and stack on small screens.

Why are media queries not needed here?

It is because grid handles responsiveness automatically.
