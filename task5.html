Task Question and Answer

Where does gap add space?

Gap adds space between grid items.

How is gap different from margin?

Gap is controlled by grid whereas margin is applied on individual items.

Does gap affect container size?

No as only the space between items increases.

Can we use gap only for rows or columns?

Yes, we can also use gap only for rows or columns

Why is gap preferred in Grid layouts?

Gap is preffered because it is a feature of Grid layouts itslef.
