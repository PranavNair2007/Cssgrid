Why do we check browser support?

We use it to make sure layout does not break on older browsers.

What happens if Grid is not supported?

If grid is not supported then browser gos back to use the normal block layout.

Is this needed for all projects?

Who uses this in real projects?

Websites, Apps, Companies, Online Games, etc.

What does safe CSS mean?

Safe CSS means it will work reliably for different versions, devices, etc.
